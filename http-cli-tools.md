# HTTP CLI tools
[TOC]

## HTTPie    
Install on Mac:
```
brew install httpie
```

## curl      
Install on Mac:
```
brew install curl
```

## wget      
Install on Mac:
```
brew install wget
```

## Curlie 	
httpie-like front-end for curl:
https://github.com/rs/curlie

Install on Mac:
Using homebrew:
```
brew install rs/tap/curlie
```
Using pkg:
```
pkg install curlie
```
